# HW5 Problem 6 - UAV Control System

Complete solution for HW5 Problem 6: UAV vertical height control using open-loop, closed-loop, and LQR controllers.

## Files

- `q6_complete.py` - Complete solution script that generates all results and plots
- `HW5_Problem6_Solution.md` - Complete documentation with theoretical derivations
- `HW5_Q6_python.md` - Python code documentation
- Individual subproblem scripts (q6_1_discretize.py, q6_2a_openloop.py, etc.)

## Usage

Run the complete solution:
```bash
python q6_complete.py
```

This generates `q6_complete_report.html` with all results and interactive plots.

## Results

- Discretization matrices A, B
- Controllability analysis
- Minimum energy open-loop control
- Closed-loop pole placement
- Robustness comparison with noise
- LQR control with three Q,R cases
